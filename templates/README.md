# Templates

Use these scaffolds to add new guardrails, workflows, prompts, or knowledge while keeping a consistent shape.

- `TEMPLATE_AGENT_NOTES.md` – constraints, auth, commands, testing limits, secrets policy.
- `TEMPLATE_WORKFLOW.md` – purpose, preconditions, ordered steps, validation, rollback.
- `TEMPLATE_PROMPT.md` – goal, audience, inputs, outputs, tone, examples.
- `TEMPLATE_KNOWLEDGE.md` – domain overview, architecture, auth, workflows, data, testing, links.
